# Security Policy

VelantisWind is a free and open-source QGIS plugin designed to support wind farm pre-assessment workflows.

Security, data protection and responsible reporting are important, especially when users may work with project layouts, wind resource data, GIS layers or confidential technical information.

---

## Supported versions

VelantisWind is currently under active development.

Until the first public release is available, only the latest version in the main repository branch is considered supported.

| Version             | Supported |
| ------------------- | --------- |
| Development version | Yes       |
| Older test versions | No        |

---

## Reporting a security issue

If you discover a security issue, please do not open a public GitHub issue.

Instead, contact the VelantisWind team directly at:

**[INSERT VELANTIS EMAIL HERE]**

Please include as much relevant information as possible:

* Description of the issue
* Steps to reproduce it
* Affected module or workflow
* QGIS version
* Operating system
* Plugin version or commit, if known
* Screenshots or logs, if useful

We will review the report and respond as soon as possible.

---

## Confidential project data

Please do not share confidential or proprietary project data in public issues, discussions or pull requests.

This includes:

* Private wind farm layouts
* Proprietary wind resource files
* Client project data
* Internal reports
* Restricted GIS layers
* Confidential coordinates or site information
* Commercial turbine data that cannot be redistributed

If a validation case requires private data, please contact the VelantisWind team separately.

---

## Responsible disclosure

We ask users and contributors to report security or confidentiality concerns responsibly.

Please avoid publicly disclosing details of a vulnerability or data-related issue until it has been reviewed and, when necessary, fixed.

---

## Scope

This security policy applies to the VelantisWind open-source plugin repository.

It does not cover third-party software, QGIS itself, external Python packages, operating system configuration or private user datasets.
