about=Experimental open-source QGIS plugin for early-stage wind-farm pre-assessment. It includes Energy/AEP workflows based on PyWake-compatible inputs, wind-turbine noise screening with fast and ISO-aligned engines, and DEM/DSM-aware shadow flicker workflows. This experimental release is intended for screening, layout comparison, design iteration and technical feedback. It is not certified regulatory or bankable assessment software. External scientific dependencies are not bundled. The Energy/AEP module requires py_wake to be installed in the same Python environment used by QGIS; Noise and Shadow Flicker can be tested independently.
